# 🚀 Rentify - Rental Management Web App

Rentify is a full-stack MERN application that allows users to list, browse, and manage rental properties efficiently. It is designed to simplify the process of renting properties for both owners and tenants.

---

## 🌐 Live Demo
👉 https://rentify-project-roan.vercel.app

---

## 📌 Features

- 🔐 User Authentication (Login/Signup)
- 🏠 Add, Edit, Delete Rental Listing
- 📄 Property Details Page
- ❤️ Wishlist / Save Properties
- 🌍 Fully Responsive Design

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Other Tools:**
- Session based Authentication
- REST APIs
- Cloudinary / Image Upload
- Deployment (Vercel / Render)
